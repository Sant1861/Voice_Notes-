# 🎤 Voice Notes App

## 📌 Overview
This is a simple Voice Notes web application that allows users to record their voice and convert it into text in real-time. The app provides an easy interface to capture, display, and edit spoken content.

---

## 🚀 Features
- 🎙 Start and Stop voice recording
- 📝 Real-time speech-to-text transcription
- ✍️ Editable text area (users can modify or delete text)
- 💰 Wallet balance display (demo)
- 📱 Responsive design (mobile, tablet, desktop)

---

## 🛠 Tech Stack
- HTML
- CSS (with media queries for responsiveness)
- JavaScript
- Browser Speech Recognition API

---

## ⚙️ How It Works
1. User clicks **Start Recording**
2. The browser accesses the microphone
3. Speech is converted into text using the Web Speech API
4. The transcribed text is displayed in the textarea
5. User can edit or delete the text manually
6. Clicking **Stop** ends the recording
